# Set-WebApp.ps1
VERSION = 0.0.1
Create WebApplication in SharePoint 2019

## Getting Started

The Set-WebApp.ps1 create a WebApplication based on file Settings.xml parameters

### Prerequisites

Set-WebApp.ps1 require file Settings.xml in folder /Config

```
C:\PS> .\Set-WebApp.ps1
```
